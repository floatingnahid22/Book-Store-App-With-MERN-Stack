## Backend

<p><b>Create a node project in Backend folder</b></p>

```sh
    npm init
```

<p><b>Then install all dependecies:</b></p>

```sh
    npm i express mongoose
```

## Frontend

<p><b> Create a react project in book-store folder: </b></p>

```sh
    npx create-react-app .
```
